## Запуск приложения:

Нужно будет nvm себе поставить, если нету 

```sh
# Склонте к себе проект.
git clone https://github.com/GeoKD/graph-visage-path.git

# Выберете папку.
cd graph-visage-path

# Установите зависимости.
npm i

# Запустите.
npm run dev
```

Приложение открывается по localhost:8080
